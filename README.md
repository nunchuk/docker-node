## Supported tags

* `7-alpine` (7.10)

## Changelist

* `PACKAGE REPOSITORY`
	Set up the package mirror server on [Alibaba Open Source Mirror Site](https://mirrors.aliyun.com/).

* `TIMEZONE`
	Change current time in [Asia/Shanghai](http://wiki.alpinelinux.org/wiki/Setting_the_timezone).
